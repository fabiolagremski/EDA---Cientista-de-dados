# análise exploratória de dados com python

## sobre o projeto

Este projeto apresenta uma análise exploratória de dados (exploratory data analysis – EDA) desenvolvida em Python utilizando a biblioteca pandas.

O objetivo foi aplicar conceitos fundamentais de análise de dados, estatística descritiva e manipulação de dados por meio da exploração de um conjunto de dados estruturado. Durante o desenvolvimento foram realizadas etapas de classificação de variáveis, cálculo de medidas estatísticas e transformação de dados para apoiar a interpretação dos resultados.

O projeto foi desenvolvido no Google Colab como parte da minha formação em ciência de dados e inteligência artificial.

## objetivos

- explorar a estrutura dos dados
- identificar diferentes tipos de variáveis
- aplicar medidas estatísticas descritivas
- manipular dados utilizando pandas
- desenvolver raciocínio analítico orientado por dados
- praticar conceitos fundamentais de análise exploratória

## atividades desenvolvidas

### exploração inicial dos dados

- visualização das primeiras observações
- identificação das colunas
- verificação dos tipos de dados
- compreensão da estrutura do conjunto de dados

### classificação de variáveis

- variáveis qualitativas nominais
- variáveis qualitativas ordinais
- variáveis quantitativas discretas
- variáveis quantitativas contínuas

### estatística descritiva

- média
- mediana
- moda

### manipulação de dados

- tratamento de valores
- conversão de tipos de dados
- criação de novas variáveis
- aplicação de funções personalizadas

### interpretação dos resultados

análise dos indicadores obtidos para compreender padrões, distribuições e características presentes nos dados.
