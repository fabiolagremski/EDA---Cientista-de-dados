# EDA---Cientista-de-dados
Análise Exploratória de Dados (EDA) com Python
Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de Análise Exploratória de Dados (EDA) utilizando Python e a biblioteca Pandas.

Ao longo do notebook são explorados diferentes conjuntos de dados para aplicação de técnicas estatísticas descritivas, identificação de tipos de variáveis e análise de indicadores relevantes para apoiar a tomada de decisão baseada em dados.

Objetivos
Compreender a estrutura dos dados
Identificar tipos de variáveis
Aplicar medidas de posição
Explorar distribuições e padrões
Desenvolver raciocínio analítico utilizando Python
Praticar manipulação de dados com Pandas
